Adolar Taggster

Version 1.3 · © PolzeSoft 2026 · polze.net · adolartaggster@polze.net

Adolar Taggster ist ein moderner MP3-Tagger mit Discogs-Integration für Windows. Tags werden korrekt in UTF-8 geschrieben — Umlaute, kyrillische und sonstige Sonderzeichen bleiben erhalten.

Installation

EXE (empfohlen)

Die Adolar Taggster.exe von der Releases-Seite herunterladen und doppelklicken — keine weitere Installation nötig.

Aus dem Quellcode

pip install PyQt6 mutagen requests Pillow librosa send2trash
python tagger.py
Hinweis: Für die Discogs-Suche empfiehlt sich ein persönlicher API-Token (60 Anfragen/Min statt 25). Token erstellen unter discogs.com → Einstellungen → Entwickler, dann unter Tools → Einstellungen eintragen.

Schnellstart

1
Ordner öffnenIm Explorer-Panel links den Künstler-Ordner aufklappen und auf den Album-Ordner doppelklicken. Die MP3s werden im Hintergrund geladen.
2
Discogs-SucheAuf Discogs klicken. Künstler und Album sind vorausgefüllt. Ein Suchergebnis anklicken lädt Details; Lade Album! öffnet den TrackMatch-Dialog.
3
Tags zuordnenIm TrackMatch-Dialog Dateinamen und Discogs-Tracks prüfen und bei Bedarf mit ▲/▼ neu ordnen.
4
Schreibe ID-TagsKlick auf Schreibe ID-Tags schreibt alle Tags, skaliert das Cover auf max. 600×600 px und speichert es als folder.jpg.
5
UmbenennenMit ✏️ Umbenennen oder dem ⚡-Button Dateien nach einer Maske umbenennen/verschieben.

Explorer

Der Explorer-Baum links zeigt Schnellzugriff-Ordner oben und darunter den vollständigen Verzeichnisbaum.

Navigation

  • Einfachklick auf einen Ordner klappt ihn auf oder zu.
  • Doppelklick auf einen Album-Ordner startet den MP3-Scan (läuft im Hintergrund, UI bleibt reaktionsfähig).
  • Während des Scans erscheint ein ✕ Abbrechen-Button in der Toolbar.

Schnellzugriff

Oben im Explorer stehen feste Ordner (Musik, Downloads, Desktop, Dokumente, Videos) sowie selbst angelegte Favoriten (⭐). Ein Klick springt direkt zu diesem Ordner im Baum.

Rechtsklick-Menü

  • ⭐ Zu Favoriten hinzufügen / entfernen — Favoriten erscheinen sofort im Schnellzugriff.
  • 🗑 In Papierkorb löschen — nach Bestätigung wird der Ordner in den Windows-Papierkorb verschoben und der Baum aktualisiert.

Dateiliste

Nach dem Doppelklick auf einen Ordner erscheinen alle gefundenen MP3s in der Tabelle — sortiert nach Album, dann nach Tracknummer (natürliche Sortierung).

Spalten

Rechtsklick auf eine Spaltenüberschrift öffnet ein Menü zum Ein-/Ausblenden. Standardmäßig sichtbar: 🖼, Track, Dateiname, Künstler, Album, Titel, Jahr, Genre, Dauer, Album Artist, BPM. Optional (Standard: aus): Bitrate, Kommentar, Label, Disc#.

Spalten können per Drag & Drop in der Überschrift umsortiert werden. Layout und Sichtbarkeit bleiben über Neustarts erhalten.

Auswahl & Rechtsklick

  • Alle / Keine in der Toolbar für Schnellauswahl.
  • Doppelklick auf eine Zeile öffnet den Tag-Editor.
  • Rechtsklick → 🏷 Tag bearbeiten — bei einer Datei: vollständiger Editor; bei mehreren: Batch-Editor.

Cover-Panel

Links unten im Explorer-Bereich werden zwei Cover nebeneinander angezeigt:

BereichInhalt

Falls eine folder.jpg existiert aber kein Tag-Cover vorhanden ist, erscheint der Button folder.jpg als Tag-Cover setzen — ein Klick schreibt das Bild als APIC-Frame in alle markierten Dateien.

Discogs-Suche

Der Suchdialog öffnet sich mit Künstler, Album und Jahr aus den Tags der ersten markierten Datei vorausgefüllt. Die Suche startet automatisch.

  • ★ Master-Releases erscheinen gold und fett — sie enthalten die vollständige Referenz-Trackliste.
  • Ein Einfachklick auf ein Ergebnis lädt Trackanzahl, Gesamtspielzeit und Cover-Status.
  • Lade Album! oder Doppelklick öffnet den TrackMatch-Dialog.
  • Künstler-Disambiguierungsnummern wie Artist (2) werden automatisch entfernt.
Tipp: Mit einem Discogs Personal Access Token (Tools → Einstellungen) erhöht sich das Limit auf 60 Anfragen/Min und Cover-Downloads werden zuverlässiger.

TrackMatch-Dialog

Zeigt lokale Dateien (links) und die Discogs-Trackliste (rechts) nebeneinander.

Zuordnung anpassen

  • Klick auf die Dateinamen-Spalte → ▲/▼ verschiebt lokale Dateien.
  • Klick auf die Discogs-Spalte → ▲/▼ verschiebt Discogs-Tracks.
  • Fehlende Dateien erscheinen als Datei nicht gefunden! in Rot.

Compilation-Alben

Enthält die Discogs-Trackliste Einträge im Format Titel /// Künstler, wird die Compilation-Option automatisch aktiviert: Titel und Track-Künstler werden getrennt geschrieben, Album Artist wird auf Various Artists gesetzt.

Cover per Drag & Drop

Ein Bild aus dem Browser oder dem Explorer kann direkt in das Cover-Feld gezogen werden. Der Button 🔍 Suche Cover Bild öffnet eine Google-Bildersuche mit Künstler + Album + Jahr.

Tag-Editor

Einzelne Datei

Doppelklick auf eine Zeile oder Rechtsklick → Tag bearbeiten öffnet den vollständigen Editor mit allen vorhandenen Werten. Nach dem Klick auf Speichern wird die Zeile in der Dateiliste aktualisiert.

Mehrere Dateien (Batch)

Mehrere Dateien markieren und dann rechtsklicken → Tag bearbeiten. Alle Felder zeigen den Platzhalter <beibehalten>. Nur ausgefüllte Felder werden geschrieben. Vor dem Speichern erscheint eine Bestätigung mit einer Übersicht aller Änderungen.

Umbenennen

Der Umbennungs-Dialog zeigt eine Vorschau die sich bei jeder Masken-Änderung sofort aktualisiert.

Masken speichern

Eigene Masken mit 💾 speichern und mit 🗑 löschen. Die zuletzt verwendete Maske wird beim nächsten Öffnen vorausgewählt.

Relative vs. absolute Pfade

  • Relative Maske %6-%2 — Dateien bleiben im selben Ordner.
  • Absolute Maske I:\Musik\%1\[%4] %3\%6-%2 — Dateien werden in die angegebene Struktur verschoben. Unterordner werden angelegt. folder.jpg wird automatisch mitkopiert.

BPM-Berechnung

Der Button ♩ BPM startet die Berechnung für alle markierten Dateien im Hintergrund. Die Statusleiste zeigt den Fortschritt. Dateien mit vorhandenem TBPM-Tag werden übersprungen.

Die Berechnung analysiert maximal 60 Sekunden pro Datei (ausreichend für genaue Ergebnisse). Der berechnete Wert wird in das TBPM-ID3-Tag geschrieben und in der BPM-Spalte der Tabelle angezeigt.

Cover-Qualitäts-Scanner

Der 🔍 Cover-Scan-Button scannt alle Unterordner des im Explorer markierten Ordners und listet Alben mit Cover-Problemen:

SymbolProblem
🔴 KorruptCover kann nicht dekodiert werden (beschädigte Datei)
🟡 Zu kleinCover kleiner als 300×300 Pixel
⚪ Kein CoverKein APIC-Tag vorhanden

Doppelklick auf einen Eintrag lädt den Ordner direkt in den Tagger zur Korrektur.

Automatische Nummerierung

Der Button # Nummerierung schreibt fortlaufende Track-Nummern in die Tags aller markierten Dateien — in der Reihenfolge wie sie in der Tabelle erscheinen.

Auffüllung richtet sich nach der Gesamtzahl:

  • 1–9 Tracks → einstellig: 1, 2, 3
  • 10–99 Tracks → zweistellig: 01, 02, 03
  • 100+ Tracks → dreistellig: 001, 002, 003

Umbenennungsmasken – Variablen

VariableBedeutungBeispiel
%1Künstler (Artist)Front 242
%2Titel (Title)Headhunter
%3AlbumFront By Front
%4Jahr1988
%5GenreEBM
%6Track-Nummer03
%tDauer04:27
%bBitrate320

Unterordner werden mit \ erzeugt. Absolute Pfade beginnen mit einem Laufwerksbuchstaben:

I:\Musik\%1\[%4] %3\%6 - %2
